# Ans::EmailReceiver

メール受信のためのクラス、モジュール

## Installation

Add this line to your application's Gemfile:

    gem 'ans-email_receiver'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ans-email_receiver
